# sidebar-window
# 🖥️ Sidebar 侧边抽屉
# Windows文本剪切板
# Windows文件中转

一个基于 Tkinter 的桌面侧边栏工具，常驻屏幕边缘，鼠标滑入即展开，提供快捷文本粘贴(拖拽粘贴)、文件粘贴(拖拽粘贴)和 AI 翻译功能。

## ✨ 功能特性

### 📌 侧边栏面板
- **自动收缩/展开**：默认以18px作为窄条贴在屏幕左/右侧，鼠标滑入展开至300px成抽屉，移出自动收缩
- **按钮列表**：自定义按钮数量不限
- **系统托盘**：关闭窗口时最小化到系统托盘，支持右键菜单恢复/退出，不显示命令行
- **实例锁**：通过文件锁保证只运行一个实例

### 🔘 自定义按钮
- **多文本随机**：每个按钮可绑定多条文本内容，点击时随机复制一条内容到剪切板，拖动到文本其他软件文本输入框可自动粘贴
- **文件粘贴**：支持为按钮绑定文件，拖动按钮到输入框可多个文件和随机文本一起粘贴(可使用Ctrl+V粘贴绑定的文件)
- **拖拽排序**：按住按钮拖动可调整顺序
- **长按删除**：长按按钮 600ms 弹出删除确认
- **拖入文件**：添加按钮时支持直接拖入文件（依赖 `windnd`）
- **同名合并**：添加同名按钮时可选择合并到已有按钮或单独创建

### 🌐 AI 翻译
- 内置文本互译功能，输入文本回车或点击AI 翻译按钮即可互译，结果自动复制到剪贴板
- 支持 30+ 翻译方向（中英、中日、中韩、中法等）
- 基于 OpenAI 兼容 API，默认使用 MiMo 模型
- 可自定义 API Key、Base URL、Model
- 注：翻译功能需自行配置API Key、Base URL、Model

### ⚙️ 配置管理
- **导出/导入配置**：一键导出按钮和设置为 JSON 文件，方便迁移
- **左/右侧切换**：可随时切换侧边栏停靠位置
- **自动持久化**：所有配置自动保存到 `sidebar_buttons.json`

## 📦 依赖

运行前需安装以下 Python 包（脚本会自动安装缺失依赖）：

| 包名 | 用途 |
|------|------|
| `pyautogui` | 模拟键盘粘贴 |
| `pyperclip` | 剪贴板读写 |
| `openai` | AI 翻译接口 |
| `pystray` | 系统托盘图标 |
| `Pillow` | 托盘图标绘制 |
| `windnd` | 文件拖拽支持 |
| `pywin32` | Windows 下实现文件粘贴 |

## 🚀 使用方法

```bash
python sidebar.py
```

启动后侧边栏以窄条形式贴在屏幕右侧，鼠标滑入即展开。

### 基本操作

| 操作 | 说明 |
|------|------|
| 鼠标滑入 | 展开侧边栏 |
| 鼠标移出 | 自动收缩 |
| 点击按钮 | 粘贴绑定的文本/文件到当前焦点窗口 |
| 拖动按钮 | 调整按钮顺序 |
| 拖动按钮到其他应用 | 自动粘贴 |
| 长按按钮 | 删除该按钮选项 |
| 输入框回车/AI翻译按钮 | 触发 AI 翻译 |

### 配置 AI 翻译

1. 点击底部 **⚙ 设置**
2. 填写 API Key、Base URL、Model
3. 选择互译语言
4. 点击保存

### 导入/导出

在设置面板中点击 **📤 导出配置** 或 **📥 导入配置**，可将按钮列表和设置保存为 JSON 文件，方便在不同机器间迁移。

## 📁 文件结构

```
sidebar.py              # 主程序

```

## 🔧 配置文件格式

`sidebar_buttons.json` 示例：

```json
{
  "buttons": [
    {
      "label": "邮件签名",
      "texts": ["Best regards,\nJohn", "此致\n张三"],
      "files": []
    },
    {
      "label": "发送报告",
      "texts": ["请查收附件"],
      "files": ["C:\\Reports\\Q1.pdf"]
    }
  ],
  "side": "right",
  "settings": {
    "api_key": "sk-xxx",
    "base_url": "https://api.xiaomimimo.com/v1",
    "model": "mimo-v2-flash",
    "translate": "中英"
  }
}
```

## ⚠️ 注意事项

- **Windows 为主**：文件粘贴功能（CF_HDROP）依赖 `pywin32`，其他平台仅支持复制文件路径，**若无法粘贴文件，请自行安装pywin32库：pip install pywin32**
- **隐藏控制台**：Windows 下启动时自动隐藏命令行窗口
- **pyautogui 安全**：`FAILSAFE` 已开启，鼠标移到左上角可紧急中断
- **API 兼容**：翻译功能使用 OpenAI SDK，兼容任何 OpenAI 格式的 API 端点
